ABCプロダクション/スタッフ紹介

制作概要

学習用制作にて、スタッフ紹介を目的とした架空のWebサイトを制作しました。
一覧・詳細ページを設け、ユーザーが人物情報を段階的に把握できる構成となっています。
パンくずリストを実装し、現在地が分かりやすいナビゲーションを構築しました。

制作のポイント

スタッフ一覧ではGridレイアウトを採用し、PC・スマートフォンの両方で見やすい配置になるよう調整しました。

工夫した点・学び

HTMLの見出し構造やセマンティックタグを意識し、可読性と保守性を考慮したコーディングを行いました。
レスポンシブ対応では、画像の有無や配置違いでもレイアウトが崩れないよう設計しています。
複数ページ構成の制作を通して、共通要素の管理やCSS設計の重要性を学びました。
実務を想定し、環境制約を考慮した実装方法を選択する意識が身についたと感じています。

使用技術

  • HTML5
  • CSS3（Flexbox / Grid）
  • JavaScript

Sneakers2 / Promotional Website

制作概要

学習用制作にて、架空のスニーカーブランドのプロモーションサイトを作成しました。

制作のポイント

ドロワーメニューを実装し、スマートフォンでも操作しやすいUIにしています。
スライダーや動画を取り入れ、視覚的に楽しめるサイトになっています。

工夫した点・学び

固定ヘッダー使用時のアンカーリンクのズレを防ぐため、 scroll-margin-top を設定しました。
レスポンシブ対応では、レイアウトの崩れが起きないよう調整を行っています。

使用技術

  • HTML5
  • CSS3（Flexbox / Grid）
  • JavaScript（jQuery）
  • slick
